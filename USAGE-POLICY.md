# Trademark Usage Policy

*Draft. This is the internal source; a sanitized version is the public
`kody-w/rapp-trademarks`. Attorney review before public use.*

The RAPP platform's code is open source under the MIT License. The **names** are
not part of that grant.

## The marks

RAPP, RAPP Brainstem, Twin in Residence, RAPP Flight Deck, and the `rapp-*`
family of names are trademarks of the RAPP project.

## What's allowed

- Use the code under the MIT License, freely.
- Say your project is "built on RAPP" or "compatible with RAPP" — accurate,
  descriptive, referential use is fine.
- Link to the official RAPP repositories.

## What's not allowed without permission

- Naming a fork or derivative "RAPP …" in a way that implies it is official or
  endorsed. **Forks must rebrand.**
- Using the marks as your own product or company name.
- Using the marks in a way that suggests sponsorship or affiliation.

## Style (the brand standard)

- Write names clean — "RAPP Brainstem", not "RAPP Brainstem™" repeated in copy.
- One trademark notice in fine print per document is enough.
- Consistent capitalization: **RAPP** (all caps), then the sub-name capitalized
  ("RAPP Brainstem", "RAPP Flight Deck").

## Reserving rights

Unregistered marks are asserted here from first use (2026-07-18 for the platform
family; earlier for `rapp-installer`). Registration status is tracked in
[`MARKS.md`](MARKS.md).
